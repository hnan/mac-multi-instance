## 简介

**多开助手**利用 macOS 原生沙盒能力，让你在 Mac 上同时运行多个微信、企业微信实例，无需破解 App、无需修改任何文件，微信或系统升级后依然可用。

适合需要同时管理多个账号的用户：工作账号 + 个人账号，或多个业务微信号。

## 功能特性

- **无限多开** — 同时运行任意数量的微信、企业微信实例
- **升级后仍可用** — 独立于 App 版本运行，无需重新配置
- **隐私安全** — 利用 macOS 沙盒创建独立实例，不修改任何 App 文件
- **轻量免费** — 安装包仅约 6MB，基础功能完全免费

## 使用方法

1. [下载安装包](https://www.heiweixin.com/releases/多开助手_v1.5.0.dmg)，拖入应用程序文件夹
2. 打开多开助手，从列表中选择要多开的应用（微信 / 企业微信）
3. 点击「启动分身」，在新实例中登录另一个账号

## 常见问题

**会被微信封号吗？**  
多开助手使用 macOS 原生沙盒机制，与破解方案不同，不修改微信文件，风险极低。

**微信或 macOS 升级后还能用吗？**  
可以。多开助手独立于 App 版本运行，升级后无需重新配置。

**需要关闭 SIP（系统完整性保护）吗？**  
不需要。多开助手完全依赖 macOS 原生能力，无需修改系统设置。

**支持 Apple Silicon（M 系列芯片）吗？**  
支持，Intel 和 Apple Silicon 均可使用。
